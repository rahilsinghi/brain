# EMNLP 2026 LaTeX Paper Scaffold for Gaze-Diffuse

This entry details the initial LaTeX paper structure scaffolded for the EMNLP 2026 conference, specifically for the [[Gaze-Diffuse]] project. It includes outlines for main sections, a bibliography, and placeholders for tables and figures, facilitating the early stages of scientific writing.

## Key Concepts

LaTeX,EMNLP (Empirical Methods in Natural Language Processing),Scientific Paper Structure,BibTeX,TikZ

## Details

A foundational LaTeX paper structure has been created for the upcoming EMNLP 2026 conference, designed for the [[Gaze-Diffuse]] project. This scaffold provides a clear starting point for authors, ensuring adherence to conference formatting and aiding in content organization.

The scaffold comprises the following key files:

*   `main.tex`: Based on the EMNLP 2023 template, this file outlines seven standard sections:
    *   Introduction
    *   Related Work
    *   Method
    *   Experimental Setup
    *   Results
    *   Analysis
    *   Conclusion + Limitations + Ethics
    Each section contains `TODO` outlines to guide content development.
*   `references.bib`: A BibTeX file pre-populated with 10 reference entries. These include foundational works and relevant research in the field, such as Sauberli, MDLM, LLaDA, [[GECO]], Diffusion-LM, MAUVE, [[GPT-2]], [[BERT]], PPLM, and readability metrics.
*   `tables/results.tex`: A placeholder for a 'methods x metrics' table, ready for `TBD` data to be inserted once experimental results are available.
*   `figures/method_overview.tex`: A TikZ diagram placeholder, providing a basic layout for illustrating the project's methodology.
