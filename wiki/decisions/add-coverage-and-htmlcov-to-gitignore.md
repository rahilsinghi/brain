# Add .coverage and htmlcov to Gitignore

This commit updates the `.gitignore` file in the `rahilsinghi/gaze-diffuse` repository to exclude Python test coverage output files, `.coverage` and the `htmlcov/` directory. This ensures that these temporary files, generated during testing, are not tracked by Git.

## Details

This chore commit, identified by SHA `5ee34e6` and authored by Rahil Singhi on 2026-03-27, adds specific entries to the `.gitignore` file within the `rahilsinghi/gaze-diffuse` repository. The purpose is to prevent test coverage artifacts from being committed to the repository.

Specifically, the following patterns were added:
*   `.coverage`: This file typically stores raw test coverage data generated by tools like `coverage.py`.
*   `htmlcov/`: This directory usually contains an HTML report of test coverage, also generated by `coverage.py`.

Excluding these files improves repository cleanliness by removing build artifacts and temporary files that are local to a developer's environment and do not contribute to the source code.
