# Gemini on Vertex AI for Repository Profile Compilation

This commit details the migration of Gemini model usage from the free AI Studio tier to a Vertex AI endpoint, leveraging $1000 in GCP billing credits. This transition eliminated rate limits, enabling the successful compilation of 10 repository profiles into wiki articles using Gemini 2.5 Flash.

## Key Concepts

Vertex AI,Gemini 2.5 Flash,GCP Credits,Rate Limits,Repository Profiles,AI Studio

## Details

The AI Studio free tier, previously used for Gemini model interactions, was replaced with a Vertex AI endpoint. This strategic move was made possible by utilizing $1000 in Google Cloud Platform (GCP) GenAI credits, primarily associated with the [[askNYC Project]]. The key benefit of this migration was the complete removal of previous rate limits, allowing for more extensive and unimpeded AI operations.

Following this change, the system successfully compiled and generated detailed wiki articles for 10 distinct repository profiles using the Gemini 2.5 Flash model. These generated articles include:
