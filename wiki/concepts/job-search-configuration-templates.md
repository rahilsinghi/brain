# Job Search Configuration Templates

This feature introduces a set of YAML configuration templates into the `Career-Datacenter` project to streamline the job search and interview preparation process. These templates define job search constraints, map keywords for specific roles, optimize for Applicant Tracking Systems (ATS), and structure behavioral interview stories.

## Key Concepts

Job Search Configuration,Interview Preparation,Applicant Tracking Systems (ATS) Optimization,Behavioral Interview Stories (STAR Method),Role-Specific Keyword Mapping,YAML Configuration

## Details

This feature, implemented in commit `bfab505` by [[Rahil Singhi]] on 2026-02-10, introduces a comprehensive set of configuration templates within the `rahilsinghi/Career-Datacenter` repository. The primary goal is to systematize and enhance both the job application and interview preparation stages. 

The following configuration files were added:
*   `preferences.yaml`: This file is designed to hold personal job search criteria, including desired job constraints, preferred company attributes, and compensation expectations. It enables a structured approach to defining ideal opportunities.
*   `role_families.yaml`: Provides detailed keyword mappings for various technical role families such as backend, data science, machine learning (ML), quantitative analysis, fullstack development, and infrastructure engineering. This helps in tailoring applications to specific job types.
*   `keyword_library.yaml`: A critical component for Applicant Tracking System (ATS) optimization, this file maps both technical and soft skills to relevant keywords, ensuring resumes and applications are more likely to pass initial automated screenings.
*   `stories_star.yaml`: A dedicated template for developing and organizing behavioral interview responses using the [[STAR Method]] (Situation, Task, Action, Result). This aids in preparing clear, concise, and impactful answers for common interview questions.

These configurations collectively facilitate systematic job tailoring and comprehensive interview preparation, aiming to improve efficiency and success rates in the job search.
