# CMDK Command Palette with Category/Source Filtering

This feature introduces a CMDK-powered command palette to the Brain-Explorer, allowing users to efficiently filter graph nodes. It enables filtering by both predefined categories and data sources, significantly enhancing navigation and discovery within the knowledge graph.

## Key Concepts

CMDK Command Palette,Category Filtering,Source Filtering,Node Filtering,User Interface (UI)

## Details

A new command palette has been integrated into the `rahilsinghi/brain-explorer` application. This palette utilizes the CMDK library to provide a fast and interactive search and filtering mechanism for the displayed knowledge graph nodes. Users can now filter nodes based on their assigned categories or their original data sources. This enhancement improves the discoverability of information and streamlines interaction with complex graph visualizations, offering a more intuitive way to narrow down relevant content.
