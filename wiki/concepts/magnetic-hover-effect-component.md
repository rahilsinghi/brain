# Magnetic Hover Effect Component

This component introduces a subtle magnetic hover effect for UI elements, primarily buttons. It causes elements to gently pull towards the user's cursor with a spring-like return motion, enhancing interactivity. The effect is designed for desktop environments only and respects user preferences for reduced motion.

A new component was added to the [[rahilsinghi/portfolio]] repository (SHA: 8c9269f, dated 2026-03-11T23:07:20Z by Rahil Singhi) that implements a magnetic hover effect.
