# Floating Section Navigation Dots

This feature introduces floating navigation dots positioned on the right side of a webpage, designed to enhance user experience on larger screens. These dots highlight the current section as the user scrolls, provide click-to-navigate functionality, and display contextual labels on hover.

## Key Concepts

Floating UI,Section Navigation,Scroll-based Highlighting,Responsive Design,User Interface (UI)

## Details

The 'Floating Section Navigation Dots' feature was integrated into the `rahilsinghi/portfolio` project.

Key functionalities of this component include:
*   **Scroll-based Highlighting**: The navigation dots dynamically highlight the corresponding section as the user scrolls through the page, visually indicating the current position.
*   **Click-to-Navigate**: Users can click on any dot to smoothly navigate directly to the associated section of the page.
*   **Hover Labels**: Descriptive labels for each section appear when a user hovers over a navigation dot, providing clear context.
*   **Responsive Design**: To maintain a clean interface on smaller devices, these navigation dots are only visible on larger screens (specifically, `lg+` breakpoints).
