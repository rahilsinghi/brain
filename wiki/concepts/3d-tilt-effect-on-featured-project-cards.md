# 3D Tilt Effect on Featured Project Cards

Implements a subtle 3D tilt effect on featured project cards within Rahil Singhi's portfolio website. This desktop-only feature provides a perspective shift based on mouse position, with a maximum 5-degree rotation, and respects user preferences for reduced motion.

## Key Concepts

3D Tilt Effect,Perspective Shift,Mouse Position Tracking,User Experience (UX) Enhancement,Reduced Motion Preference

## Details

This feature introduces an interactive 3D tilt effect to the featured project cards on Rahil Singhi's portfolio website. The effect creates a subtle perspective shift, making the cards appear to rotate slightly in response to the user's mouse movement over them.

**Key Characteristics:**
*   **Interaction:** The tilt angle dynamically adjusts based on the mouse's position relative to the card.
*   **Maximum Rotation:** The rotation is capped at a maximum of 5 degrees, ensuring the effect remains subtle and non-distracting.
*   **Target Device:** The effect is designed specifically for desktop environments, avoiding potential issues or performance impacts on mobile devices.
*   **Accessibility:** It respects the user's `prefers-reduced-motion` operating system setting, automatically disabling the effect for users who prefer less motion, thus enhancing accessibility.
