# RhymeSheet UI: Color-Coded Lyrics Panel

The RhymeSheet UI introduces a dynamic, color-coded lyrics panel designed for the Raag project. It features individual rhyme highlights and can operate in both synced and static display modes, providing an enhanced lyrical analysis experience.

## Key Concepts

RhymeSheet UI,Color-coded lyrics,Synced Mode,Static Mode,RhymeBar,RhymeSheet,RhymeSheetPanel

## Details

The RhymeSheet UI provides a sophisticated interface for displaying and analyzing lyrics within the [[Raag Project]]. Its core functionality revolves around color-coding rhymes, making lyrical patterns visually intuitive.

Key components include:
*   **RhymeBar:** An individual component responsible for rendering a single bar of lyrics. It applies colored background highlights to words identified as rhymes, matching a predefined reference style.
*   **RhymeSheet:** This is the main scrollable lyrics display area. It organizes lyrics by section and includes a legend to explain the color-coding scheme.
*   **RhymeSheetPanel:** A slide-up glass panel that houses the RhymeSheet. This panel features a toggle allowing users to switch between two primary display modes:
    *   **Synced Mode:** Likely synchronizes lyric display with audio playback.
    *   **Static Mode:** Displays lyrics independently of playback, allowing for manual scrolling and review.

The implementation, led by [[Rahil Singhi]] with contribution from [[Claude Opus 4.6]], involved significant additions to the codebase, enhancing the user's ability to engage with lyrical content through visual cues.
