# EMNLP 2026 LaTeX Paper Scaffold for Gaze-Diffuse

This entry documents the initial scaffolding of a LaTeX paper structure for the EMNLP 2026 conference within the `gaze-diffuse` repository. It includes a `main.tex` based on the EMNLP 2023 template, a `references.bib` with key entries, and placeholders for results tables and method diagrams, providing a foundational framework for the research publication.

## Details

A foundational LaTeX paper structure has been established in the [[Gaze-Diffuse]] repository, targeting the EMNLP 2026 conference. This scaffold, commit `6155603` by [[Rahil Singhi]] on 2026-03-15, introduced 500 new lines across four files, creating a comprehensive starting point for manuscript development.

Key components include:
*   **`main.tex`**: Configured using the EMNLP 2023 template, this file outlines the standard seven sections of a research paper: Introduction, Related Work, Method, Experimental Setup, Results, Analysis, and Conclusion (+ Limitations + Ethics). Each section contains `TODO` outlines to guide content creation.
*   **`references.bib`**: A bibliography file populated with 10 essential entries, including foundational works such as Sauberli, MDLM, LLaDA, [[GECO]], Diffusion-LM, MAUVE, GPT-2, BERT, PPLM, and general readability metrics. This provides a strong starting point for citing relevant literature.
*   **`tables/results.tex`**: A placeholder for the main results table, structured to accommodate various methods and metrics (`methods x metrics table with TBD placeholders`).
*   **`figures/method_overview.tex`**: A TikZ diagram placeholder, complete with a basic layout, ready for the detailed visualization of the research methodology.
