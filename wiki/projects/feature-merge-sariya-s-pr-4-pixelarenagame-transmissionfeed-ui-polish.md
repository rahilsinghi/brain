## Details

This merge (SHA `5ace672`) from the `frontend/new` branch into the main `rahilsinghi/karen` repository incorporates substantial updates and new functionalities, resolving previous conflicts. The changes primarily focus on enhancing the frontend user experience and introducing new gameplay elements.

**Key Additions:**
*   **PixelArenaGame:** A pixel-art based arena game, replacing the previous `KarenGameMode`.
*   **TransmissionFeed:** A new component for displaying interactive feeds.
*   **CrabPlayer, ChannelArtifact, TargetSprite:** New game-related entities.
*   **`useArenaGameLoop` hook:** A custom React hook to manage the game loop logic.

*   **Deep Linking:** Implemented with demo fill functionality on the trigger page to enhance navigation and direct access.

*   Context-sensitive input fields.
*   Personality button styling.
*   The `CommentaryLog` component.
*   The `LevelTimeline` component.

**Escalation Page:**
