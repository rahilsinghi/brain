# LinkedIn Outreach Campaign — Hospitality Tech Startups (Mar 26–31, 2026)

A structured LinkedIn outreach campaign targeting founders, CTOs, and key decision-makers at seven hospitality tech and AI travel startups during the week of March 26–31, 2026. The campaign leverages the sender's internship experience at Kismet building MCP tools, PMS adapters, and LLM extraction pipelines to establish relevance with each target company. The goal is to secure informational conversations and explore full-time engineering roles following an NYU MS Computer Engineering graduation in May 2026.

## Key Concepts

- **MCP Tools**: Model Context Protocol tools (searchHotels, getHotelRates, bookHotelOffer) built at Kismet using NestJS and JSON-RPC over SSE, enabling AI agents to search and book hotel inventory
- **PMS Adapters**: Integrations with Property Management Systems including OHIP/Oracle Opera and Mews, built with asyncio parallelization and Redis caching
- **Amadeus GDS Integration**: Rate lookup pipelines using the Amadeus Global Distribution System with PMS-fallback logic
- **Playwright + LLM Extraction Pipeline**: Automated hotel data extraction pipeline using Playwright and GPT-4o achieving >95% field accuracy across 10K+ hotel pages/month
- **Booking Engine Coverage**: Experience across 7–8 booking engines including SynXis, iHotelier, Allora, and AZDS
- **BigQuery Analytics**: Data warehouse with multi-signal attribution across hotel booking flows
- **Connection Note Format**: LinkedIn connection notes limited to 300 characters, tailored to each recipient's background and company focus
- **DM Format**: Longer LinkedIn direct messages sent after connection acceptance, including personal background, specific technical overlap, and a 15-minute meeting request
