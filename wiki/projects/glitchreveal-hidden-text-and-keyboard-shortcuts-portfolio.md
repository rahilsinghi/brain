# GlitchReveal Hidden Text and Keyboard Shortcuts (Portfolio)

This commit introduces two new interactive features to the `rahilsinghi/portfolio` project: GlitchReveal, which displays hidden text with a scrambling effect upon cursor proximity, and a set of keyboard shortcuts for quick navigation and access to portfolio functionalities. These enhancements improve user engagement and accessibility.

## Key Concepts

GlitchReveal Effect,Keyboard Shortcuts,Interactive UI,User Experience

## Details

This feature adds an engaging visual effect called `GlitchReveal` to the portfolio. Text configured with GlitchReveal remains invisible until the user's cursor approaches within 100 pixels, at which point it dynamically scrambles into readability. This adds a subtle, interactive discovery element to the site.

Simultaneously, a new set of `Keyboard Shortcuts` has been implemented to streamline navigation and access:
*   `Cmd+K` (or `Ctrl+K` on Windows/Linux) opens an interactive terminal interface, potentially related to existing [[Add 8 Easter Egg Terminal Commands to Portfolio]].
*   `?` displays a cheat sheet detailing all available keyboard commands.
*   `g→h`: Navigates to the GitHub profile.
*   `g→l`: Navigates to the LinkedIn profile. This could be related to broader initiatives like [[Career-Datacenter: LinkedIn Follow-Up and Hackathon Portfolio Update]].
*   `g→r`: Navigates to the resume, likely referencing the [[Original LaTeX Resume Source]] or generated resume documents.
