# System Design Mastery Learning Platform

This article details the initial development of a lightweight, interactive learning platform for system design, ranging from basics to interview mastery. Built with Vanilla HTML, CSS, and JavaScript, it features a dark-mode UI, skill-tree progress tracking, a hash-based SPA router, and a quiz engine with confetti.

## Key Concepts

Interactive Learning Platform,System Design,Vanilla HTML/CSS/JS,Single Page Application (SPA),Quiz Engine,Progress Tracking,Skill-Tree UI,Dark Mode UI,Syntax Highlighting (Prism.js)

## Details

This commit introduces the initial version of a lightweight, interactive learning platform designed to teach system design concepts from basics to interview mastery. The platform is built using Vanilla HTML, CSS, and JavaScript, requiring no build step for simplicity.

Key features include:
*   **User Interface**: A dark-mode UI with a skill-tree landing page for visual progress tracking.
*   **Navigation**: A hash-based Single Page Application (SPA) router for smooth transitions.
*   **Interactivity**: A quiz engine with confetti animations for completion, and a scrollspy sidebar for content navigation.
*   **Content Structure**: Module 1, titled "What is System Design?", is complete. It features story-based content, 3 interactive SVG diagrams, and code examples, alongside 5 quiz questions.
*   **Code Presentation**: Integrates Prism.js for syntax highlighting across multiple languages including Python, JavaScript, SQL, Go, Bash, JSON, and YAML.
*   **Curriculum**: An 11-module curriculum is scaffolded, with content for modules 2-11 planned for future development.

The platform aims to provide an engaging and self-contained environment for learning complex system design principles.
