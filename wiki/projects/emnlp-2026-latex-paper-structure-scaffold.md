# EMNLP 2026 LaTeX Paper Structure Scaffold

This content describes the initial LaTeX paper structure scaffold for the EMNLP 2026 conference, part of the `gaze-diffuse` repository. It includes the `main.tex` document using the EMNLP 2023 template with seven sections outlined, a `references.bib` file with ten initial entries, and placeholder files for results tables and method overview figures.

## Key Concepts

LaTeX,EMNLP Conference,Scientific Paper Structure,Bibliography Management,TikZ Diagrams

## Details

A commit (`6155603`) on 2026-03-15 by Rahil Singhi in the `rahilsinghi/gaze-diffuse` repository established a foundational LaTeX paper structure for the EMNLP 2026 conference. This scaffold involved changes across four files, adding 500 lines of content.

Key components introduced:

*   `main.tex`: This file serves as the primary document, utilizing the EMNLP 2023 LaTeX template. It is structured into seven standard sections, each with `TODO` outlines to guide content creation: Introduction, Related Work, Method, Experimental Setup, Results, Analysis, and a combined Conclusion + Limitations + Ethics section.
*   `references.bib`: A bibliography file pre-populated with 10 initial entries. These references include works such as Sauberli, MDLM, LLaDA, [[GECO]], Diffusion-LM, MAUVE, [[GPT-2]], [[BERT]], PPLM, and general readability metrics, providing a starting point for the paper's literature review.
*   `tables/results.tex`: A placeholder LaTeX file intended for the results table. It includes a methods-by-metrics table with `TBD` placeholders, indicating where specific experimental outcomes will be recorded.
*   `figures/method_overview.tex`: This file contains a placeholder for a TikZ diagram, designed to illustrate the paper's methodology. The current content establishes the layout for future diagram implementation.
