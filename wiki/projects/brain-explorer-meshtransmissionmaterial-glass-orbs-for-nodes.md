# Brain-Explorer: MeshTransmissionMaterial Glass Orbs for Nodes

This update enhances the visual representation of nodes within the Brain-Explorer interface by upgrading them to render as glass orbs using Three.js's `MeshTransmissionMaterial`. This change aims to provide a more sophisticated and visually appealing aesthetic for knowledge graph elements.

[[MeshTransmissionMaterial]],Node Visualization,[[Three.js]]

## Details

The `Brain-Explorer` project received an update (SHA: `318c1d9`) on April 12, 2026, by Rahil Singhi, which significantly alters the visual styling of its nodes. Previously, nodes might have been represented using simpler materials. This commit specifically transitions node rendering to utilize `MeshTransmissionMaterial` from the Three.js library.

`MeshTransmissionMaterial` is an advanced material in Three.js that simulates transparent objects like glass, allowing for realistic refraction and transmission effects. By applying this material, the nodes now appear as sophisticated glass orbs, likely improving the overall aesthetic and user experience of the knowledge graph visualization.

This enhancement involved changes to one file, with 16 lines added and 2 lines removed, indicating a focused material upgrade.
