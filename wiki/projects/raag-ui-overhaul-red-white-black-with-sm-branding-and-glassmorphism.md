# Raag UI Overhaul: Red/White/Black with SM Branding and Glassmorphism

This wiki article details a comprehensive frontend redesign for the Raag project, implementing a striking red, white, and black color palette inspired by Seedhe Maut branding. The overhaul features glassmorphism cards with premium hover effects, custom typography, and integrated brand elements like the SM logo as an assistant avatar.

## Key Concepts

- User Interface (UI) Design,- Branding (Seedhe Maut),- Glassmorphism,- Frontend Development,- Typography (KinetikaMaut, TTCommonsPro),- UI/UX Features (album art carousel, score rings, energy bars)

## Details

This commit (`1eeaec7`) by [[Rahil Singhi]] on March 4, 2026, represents a complete overhaul of the Raag project's frontend user interface. The redesign centers around a distinct red, white, and black color scheme, directly referencing the [[Seedhe Maut]] brand palette (specifically `#d91d1c` for red and `#050505` for black). 

Key design elements include:
- **Typography**: Headings utilize the `KinetikaMaut` display font, while body text employs `TTCommonsPro`.
- **Glassmorphism**: UI cards feature a glassmorphism style, enhanced with premium hover effects for an interactive experience.
- **Brand Integration**: The [[Seedhe Maut]] logo is used as an assistant avatar, and album art carousels are implemented for empty states.
- **Detailed UI Elements**: Album thumbnails are displayed in song cards, and score rings and energy bars are accented with the signature red color.

This overhaul involved significant code changes, with 634 additions and 258 deletions across 9 files.
