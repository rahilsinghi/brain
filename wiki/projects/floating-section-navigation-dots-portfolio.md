# Floating Section Navigation Dots (Portfolio)

This feature adds floating navigation dots to the right side of the portfolio website, enhancing user experience by indicating the current section on scroll and allowing direct navigation. Hovering over the dots reveals labels for each section, with the component designed to be responsive and visible only on larger screens.

## Key Concepts

Floating Navigation,Scroll-Based Highlighting,Click-to-Navigate UI,Hover Effects,Responsive Web Design,User Interface (UI) Enhancements

## Details

This feature introduces a set of floating navigation dots positioned on the right side of the portfolio website. These dots serve as visual indicators for the user's current scroll position, automatically highlighting the active section as the user scrolls through the page. Each dot is clickable, enabling quick navigation to its corresponding section.

To improve usability, labels appear when a user hovers over a dot, providing context for the destination section. The component is designed with responsive principles, ensuring it is only visible on large screens (defined as `lg+`) and automatically hidden on mobile devices to optimize the user experience for smaller viewports.
