# Floating Section Navigation Dots

This feature introduces floating navigation dots to the portfolio website, enhancing user experience by indicating the current section on scroll and allowing direct navigation. Labels for sections appear on hover, and the feature is designed to be visible only on larger screens to maintain responsiveness.

## Key Concepts

Floating navigation,UI/UX,Scroll-based highlighting,Responsive design

## Details

This enhancement (commit `03fddc7`) adds floating navigation dots to the right side of the [[rahilsinghi/portfolio]] website. These dots serve several purposes:

*   **Current Section Highlighting**: As the user scrolls through the website, the corresponding navigation dot automatically highlights, providing a clear visual cue of their current position.
*   **Direct Navigation**: Users can click on any dot to smoothly navigate to the associated section of the page.
*   **Hover Labels**: To improve usability, descriptive labels for each section appear when a user hovers over a navigation dot.
*   **Responsive Design**: The feature is implemented with responsiveness in mind, appearing only on large (`lg+`) screen sizes and being hidden on mobile devices to prevent UI clutter.
