# Vercel Analytics, Speed Insights, and JSON-LD Integration (Portfolio)

This update integrates Vercel Analytics and Speed Insights into the `rahilsinghi/portfolio` repository to monitor website performance and user engagement. It also adds JSON-LD structured data and Open Graph (OG) image metadata for improved Search Engine Optimization (SEO) and social media sharing.

## Details

This feature addition, captured in commit `06400a8` of the `rahilsinghi/portfolio` repository on 2026-03-11, enhances the portfolio's technical infrastructure. The primary goals were to improve website monitoring, performance, and discoverability.

Key changes include:
*   **Vercel Analytics**: Integration of Vercel's analytics solution to track website traffic, user behavior, and other engagement metrics.
*   **Vercel Speed Insights**: Implementation of Vercel's performance monitoring tool to identify and optimize load times and overall website speed.
*   **JSON-LD Structured Data**: Addition of JSON-LD (JavaScript Object Notation for Linked Data) to embed structured data directly into the HTML. Specifically, a Person schema was added to provide search engines with rich, semantically meaningful information about the portfolio owner, improving search result presentation.
*   **Open Graph (OG) Image Metadata**: Inclusion of Open Graph tags to control how content appears when shared on social media platforms, ensuring visually appealing and informative previews.

These changes collectively aim to boost the portfolio's SEO, enhance its presentation across various platforms, and provide valuable insights into its performance.
