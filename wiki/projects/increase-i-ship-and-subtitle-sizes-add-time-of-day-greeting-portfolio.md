# Increase 'I Ship' and Subtitle Sizes, Add Time-of-Day Greeting (Portfolio)

This commit enhances the user interface of the `rahilsinghi/portfolio` by increasing the 'I Ship' heading and subtitle font sizes for improved readability. Additionally, it introduces a dynamic, contextual greeting based on the visitor's local time, adding a personalized touch to the website.

## Key Concepts

Responsive Typography,Dynamic Content,Time-based Personalization,Frontend Development

## Details

This feature (`feat: increase I Ship and subtitle sizes, add time-of-day greeting`) was implemented in the [[rahilsinghi/portfolio]] repository with commit `ae7e58a` on March 12, 2026. The changes primarily involve UI adjustments to the main headings and the addition of a new personalization element.

Specifically, the 'I Ship' heading's responsive font size was increased from `clamp(1.8rem, 4.5vw)` to `clamp(2.2rem, 5.5vw)`. Similarly, the subtitle's font size was adjusted from `clamp(1rem, 2.5vw)` to `clamp(1.2rem, 2.8vw)`, enhancing its prominence and readability across different screen sizes.

A key new feature is the integration of a contextual greeting that changes based on the visitor's local time. This adds a layer of personalization to the user experience, making the portfolio feel more interactive and welcoming.
