# Ambient Auto-Rotation with Idle Resume (Brain-Explorer)

This feature introduces an ambient auto-rotation capability to the camera in the Brain-Explorer application. The camera automatically rotates when idle, pausing the rotation when user interaction is detected and resuming after a period of inactivity.

## Key Concepts

Ambient Auto-Rotation,Idle Resume,Camera Control,User Experience,Brain-Explorer

## Details

This commit introduces a new camera behavior for the `rahilsinghi/brain-explorer` project. The camera now features ambient auto-rotation, which engages when the user is idle. Upon detecting user interaction, the auto-rotation pauses and then resumes automatically after a specified period of inactivity. This enhancement aims to improve the visual engagement and dynamic presentation of the Brain-Explorer graph interface.
