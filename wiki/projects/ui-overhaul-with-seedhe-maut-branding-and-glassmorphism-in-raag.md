# UI Overhaul with Seedhe Maut Branding and Glassmorphism in raag

This commit details a comprehensive frontend redesign for the [[raag]] project, incorporating the distinct red, white, and black color palette of the Seedhe Maut brand. The overhaul introduces a glassmorphism design language with premium hover effects, alongside custom typography and themed UI elements such as an album art carousel and score rings, significantly enhancing the visual identity and user experience.
