# GlitchReveal and Keyboard Shortcuts Feature

This commit introduces two new features to the `rahilsinghi/portfolio` project: the GlitchReveal effect for interactive hidden text and a set of custom keyboard shortcuts. These additions aim to enhance the user experience by providing dynamic content revelation and efficient site navigation.

## Details

This feature implementation, tracked by commit `5c40080` on `2026-03-12` by [[Rahil Singhi]], adds significant enhancements to the [[rahilsinghi/portfolio]] website.

### GlitchReveal

The **GlitchReveal** is an interactive UI element designed to present "invisible text" that dynamically scrambles and becomes visible when the user's cursor approaches within a 100-pixel radius. This effect adds an engaging and subtle layer of interaction, encouraging exploration and discovery on the portfolio site.

### Keyboard Shortcuts

A new set of **Keyboard Shortcuts** has been integrated to improve site navigation and accessibility:

-   `Cmd + K`: Opens a terminal-like interface, presumably for quick commands or search.
-   `?`: Displays a cheat sheet, providing users with a quick reference to available shortcuts and functionalities.
-   `g → h`: Navigates to [[GitHub]].
-   `g → l`: Navigates to [[LinkedIn]].
-   `g → r`: Navigates to the user's [[Resume]].
