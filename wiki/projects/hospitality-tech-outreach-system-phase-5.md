# Hospitality Tech Outreach System (Phase 5)

This project phase, completed on February 25, 2026, focused on developing and implementing a fully automated outreach system for hospitality technology companies. ## Details

Phase 5 of the Career Datacenter project involved the creation of a comprehensive hospitality tech outreach system. The system automates various aspects of professional outreach, making the process efficient and personalized.

**Key Features Implemented:**
*   **Email Generation:** Utilizes an LLM (Haiku) for generating category-aware emails based on predefined templates.
*   **Contact Management & Tracking:** A robust system to manage and track the status of contacts.
*   **LinkedIn Message Drafting:** Automated drafting of personalized messages for LinkedIn outreach.
*   **Follow-up Automation:** Ensures timely and structured follow-up communications.

**Companies Processed:** The system was used to process outreach for five hospitality technology companies:
*   Mews
*   Canary
*   Actabl
*   Duetto
*   HiJiffy
