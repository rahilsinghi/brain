# Image Carousel for Hackathon Projects with Two-Column Layout

This feature introduces a visually appealing and functional image carousel to the hackathon projects section of the portfolio. It is designed with a two-column layout, allowing project content to reside on the left while the interactive carousel occupies the right side.

## Key Concepts

Image Carousel,Two-Column Layout,Arrow Navigation,Crossfade Animation,Dot Indicators,Gradient Border Frame,Sticky Positioning

## Details

This commit (`9f793b9`) from the `rahilsinghi/portfolio` repository by [[Rahil Singhi]] implements an image carousel specifically for showcasing hackathon projects. The design integrates a two-column layout, where the primary project content is displayed in the left column, and the compact image carousel resides on the right.

The carousel features:
*   **Navigation:** Intuitive arrow controls for browsing through images.
*   **Animation:** A smooth crossfade animation between images.
*   **Indicators:** Dot indicators provide visual feedback on the current image position.
*   **Visual Styling:** A distinctive gradient border frame and green corner brackets enhance the carousel's aesthetic.
*   **Layout Behavior:** Images within the carousel are sticky-positioned on desktop, ensuring they remain visible while the user scrolls through the accompanying project description. This enhancement improves the presentation and user experience for detailed project showcases, particularly for projects like [[Ask NYC]] or the [[Karen Project]] which may be featured in the hackathon section.
