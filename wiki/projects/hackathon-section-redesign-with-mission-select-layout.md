# Hackathon Section Redesign with Mission Select Layout

This commit redesigns the hackathons section of the portfolio website, transitioning from a vertical card list to an interactive horizontal 'Mission Select' layout. The new design incorporates thumbnail previews, a sliding active indicator, staggered content reveals, an image carousel with corner brackets, and terminal-style metadata labels.

## Details

The hackathon section of the [[Portfolio]] website underwent a significant redesign, replacing the previous vertical card list with an engaging 'Mission Select' layout. This new structure features a horizontal card selector that allows users to browse through hackathon projects. Upon selection, an expanded detail view is presented, enriched with various interactive elements.

Key enhancements include thumbnail previews for each project, a sliding active indicator to highlight the current selection, and staggered content reveals that progressively display information. An image carousel with distinctive corner brackets provides a visually appealing way to showcase project screenshots and images. Additionally, terminal-style metadata labels, such as `DEPLOYMENT_ROLE` and `STACK_VERIFICATION`, are used to present project details, adding a thematic touch. The redesign specifically incorporates images from the [[Flock]] hackathon and a screenshot from the [[MarkPush]] project.
