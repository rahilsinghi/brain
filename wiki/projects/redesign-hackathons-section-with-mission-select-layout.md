# Redesign Hackathons Section with Mission Select Layout

This commit overhauls the Hackathons section of the rahilsinghi/portfolio, replacing a vertical card list with an interactive, horizontal 'Mission Select' layout. The redesign introduces thumbnail previews, a sliding active indicator, staggered content reveals, an image carousel, and terminal-style metadata labels for an enhanced user experience.

## Details

The hackathons section within the [[rahilsinghi/portfolio]] repository was significantly redesigned. The previous vertical card list structure was replaced with a more dynamic and engaging horizontal card selector, inspired by a 'Mission Select' layout often found in video games or interactive experiences.

Key features introduced with this redesign include:
*   **Thumbnail Previews**: Small visual representations for each hackathon project, allowing for quick scanning.
*   **Sliding Active Indicator**: A visual cue that highlights the currently selected project in the horizontal list.
*   **Staggered Content Reveals**: Elements within the detailed view animate into place sequentially, creating a smoother and more engaging reveal.
*   **Image Carousel with Corner Brackets**: A dedicated component to showcase multiple images or screenshots for an individual project, framed by a distinctive design.
*   **Terminal-Style Metadata Labels**: Project metadata, such as `DEPLOYMENT_ROLE` and `STACK_VERIFICATION`, is presented using a command-line interface aesthetic, adding a unique stylistic touch.

The update incorporates specific project visuals, including images from the [[Flock Project]] hackathon and a screenshot of the [[MarkPush]] project, further enriching the portfolio's content.
