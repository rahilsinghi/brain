# Image Carousel and Two-Column Layout for Hackathon Projects

This feature introduces an image carousel and a two-column layout for hackathon projects within the portfolio. The carousel offers arrow navigation, crossfade animations, and dot indicators, framed with a gradient border and green corner brackets. This design allows project content to occupy the left column while images are sticky-positioned on the right for desktop users.

## Key Concepts

Image Carousel,Two-Column Layout,Arrow Navigation,Crossfade Animation,Dot Indicators,Gradient Border,Sticky Positioning,User Interface (UI) Enhancement

## Details

A new feature has been implemented for hackathon projects in the [[Portfolio]] repository (`rahilsinghi/portfolio`, SHA: `9f793b9`). This update introduces a compact image carousel on the right side of the project display. The carousel includes intuitive arrow navigation for cycling through images, smooth crossfade animations between slides, and clear dot indicators to show the current image position. Visually, it is framed with a gradient border and distinctive green corner brackets, enhancing its aesthetic appeal.

Simultaneously, a two-column layout has been adopted for these project pages. This layout ensures that the main textual content of the project fills the left column, providing ample space for descriptions and details. On desktop displays, the images within the carousel are designed to be sticky-positioned, ensuring they remain visible as the user scrolls through the project's textual content in the left column. This enhancement significantly improves the presentation and user experience for showcasing hackathon projects.
