# Hero Parallax Layers and Aurora Gradient Background (Portfolio)

This feature introduces dynamic parallax scrolling layers to the hero section and an animated, low-opacity aurora gradient background, enhancing the visual depth and aesthetic appeal of the portfolio site.

## Key Concepts

[[Parallax Scrolling]],[[CSS Gradients]],[[UI/UX Design]],[[Frontend Animation]]

## Details

This commit (`606b934`) to the [[rahilsinghi/portfolio]] repository implements two key visual enhancements for the hero section:

1.  **Parallax Layers**: Text elements within the hero section now move at varying speeds when the user scrolls, creating a sense of depth and a dynamic parallax effect.
2.  **Aurora Gradient Background**: An animated gradient mesh, designed to resemble an aurora, has been added behind the hero text. This background is rendered with low opacity and is implemented entirely using CSS, contributing to a subtle yet engaging visual experience.
