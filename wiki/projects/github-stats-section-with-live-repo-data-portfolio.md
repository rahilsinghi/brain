# GitHub Stats Section with Live Repo Data (Portfolio)

This feature introduces a new section to the rahilsinghi/portfolio website, dynamically displaying live repository statistics fetched from the GitHub API. It uses server components with 12-hour revalidation to showcase top repositories, including details like stars, forks, language, and descriptions.

## Key Concepts

[[GitHub API]],[[Server Components]],[[Data Revalidation]],[[Dynamic Data Display]]

## Details

A new GitHub stats section has been implemented on the [[rahilsinghi/portfolio]] website. This section is powered by a server component responsible for fetching real-time repository data directly from the [[GitHub API]]. To ensure data freshness without excessive API calls, the component is configured with a 12-hour revalidation period.

The displayed statistics highlight top repositories and include key metrics such as:
*   Number of stars
*   Number of forks
*   Primary programming language
*   Repository description

This enhancement provides visitors with an up-to-date overview of the author's most prominent open-source contributions directly on the portfolio page.
