# Portfolio UI Enhancements: Scaled Text and Dynamic Time-of-Day Greeting

This commit introduces user interface enhancements to the personal portfolio, specifically increasing the size of key text elements ('I Ship' and subtitle) for better readability. It also adds a dynamic, contextual greeting that adjusts based on the visitor's local time, contributing to a more personalized user experience.

## Key Concepts

UI/UX Design,Responsive Text Scaling,Dynamic Content Generation,Frontend Development

## Details

The commit `ae7e58a` in the [[rahilsinghi/portfolio]] repository focuses on refining the visual presentation and user interaction of the portfolio website. The 'I Ship' heading text size was increased from `clamp(1.8rem, 4.5vw)` to `clamp(2.2rem, 5.5vw)`, while the subtitle text was scaled from `clamp(1rem, 2.5vw)` to `clamp(1.2rem, 2.8vw)`. These adjustments aim to improve the hierarchy and readability of the main page elements.

Additionally, a new feature was implemented to display a contextual greeting message. This greeting dynamically changes based on the visitor's local time, offering a personalized touch (e.g., 'Good morning', 'Good afternoon', 'Good evening'). This enhancement contributes to a more engaging and welcoming user experience on the [[Portfolio]].
