# HTTPS Dev Server, mkcert, and NLP Scripts Integration (raag)

This commit to the `raag` repository introduces support for an HTTPS development server using `mkcert` certificates, enhancing local development security. It also adds necessary CORS origins and two utility NLP scripts for batch ingestion and processing. This improves the development environment and provides new tooling for natural language processing tasks within the project.
