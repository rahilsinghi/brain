# Portfolio Feature: GlitchReveal Hidden Text and Keyboard Shortcuts

This feature introduces 'GlitchReveal' hidden text, which becomes animated and visible when the cursor approaches within a 100px radius. It also adds global keyboard shortcuts to the portfolio, allowing quick access to a terminal, a cheat sheet, and direct navigation to GitHub, LinkedIn, or the resume.

## Key Concepts

[[GlitchReveal]],[[Keyboard Shortcuts]],[[Portfolio]],[[Terminal Easter Eggs]],[[UI/UX Enhancements]]

## Details

This commit enhances the user experience of the portfolio by adding two distinct features:

1.  **GlitchReveal Hidden Text**: This stylistic effect applies to certain text elements, rendering them initially invisible. When the user's cursor comes within a 100-pixel proximity, the hidden text 'scrambles' to life with a glitch animation, revealing its content. This adds an interactive and engaging element to the portfolio.

2.  **Global Keyboard Shortcuts**: For improved navigation and access, the portfolio now responds to several keyboard commands:
    *   `Cmd + K`: Opens a terminal-like interface within the portfolio.
    *   `?`: Displays a cheat sheet outlining all available keyboard shortcuts.
    *   `g + h`: Navigates directly to the author's GitHub profile.
    *   `g + l`: Navigates directly to the author's LinkedIn profile.
    *   `g + r`: Navigates directly to the author's resume.
