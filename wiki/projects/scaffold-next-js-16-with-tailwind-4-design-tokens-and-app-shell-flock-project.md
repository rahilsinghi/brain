# Scaffold Next.js 16 with Tailwind 4, Design Tokens, and App Shell (Flock Project)

This commit initiates the scaffolding for the [[Flock Project]], setting up a Next.js 16 application. It integrates Tailwind CSS 4, establishes a design token system, and lays out the foundational app shell.
Key dependencies like AI SDKs, Supabase SSR, Clerk for authentication, ElevenLabs, and Framer Motion are installed, alongside configurations for Turbopack, PostCSS, TypeScript paths, and a dark theme.
