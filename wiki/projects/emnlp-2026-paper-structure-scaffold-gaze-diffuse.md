# EMNLP 2026 Paper Structure Scaffold (gaze-diffuse)

This entry documents the initial LaTeX paper structure scaffold for a project housed in the `rahilsinghi/gaze-diffuse` repository, aimed at the EMNLP 2026 conference. It includes a `main.tex` based on the EMNLP 2023 template with seven outlined sections, a `references.bib` with ten key entries, and placeholders for tables and a TikZ-generated method overview figure.

## Details

This commit (`6155603`) by Rahil Singhi on 2026-03-15 initiated the foundational LaTeX structure for an EMNLP 2026 submission within the `rahilsinghi/gaze-diffuse` repository. The commit involved adding 500 lines across 4 files to establish the paper's framework.

Key components include:

*   `main.tex`: This primary document utilizes the EMNLP 2023 template and is pre-structured with seven distinct sections, each marked with `TODO` outlines to guide content development. The sections are: Introduction, Related Work, Method, Experimental Setup, Results, Analysis, and Conclusion (which also incorporates Limitations and Ethics statements).
*   `references.bib`: A bibliography file populated with 10 initial entries, crucial for grounding the research. These include: Sauberli, MDLM, LLaDA, GECO, Diffusion-LM, MAUVE, GPT-2, BERT, PPLM, and works on readability metrics.
*   `tables/results.tex`: A placeholder LaTeX file designed for a results table, anticipating a 'methods x metrics' structure with `TBD` fields to be populated upon experimental completion.
*   `figures/method_overview.tex`: This file contains a basic layout for a TikZ diagram, intended to visually represent the project's methodology once developed.
