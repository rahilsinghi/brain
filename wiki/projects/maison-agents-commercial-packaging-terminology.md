# Maison Agents — Commercial Packaging & Terminology

Maison Agents is a modular platform of AI agents designed to streamline hotel commercial operations. The platform uses a three-layer technical model (Tool, Skill, Agent) and offers pre-built, custom, and mini-agents for various hospitality tasks.

## Key Concepts

- Three-Layer Technical Model
- Pre-built Agents
- Custom Agents
- Mini-agents
- Commercial Vocabulary for Hospitality AI

## Details

### Three-Layer Technical Model

| Layer | Term | Example |
|---|---|---|
| Bottom | Tool | PMS lookup, WhatsApp send, CRM update |
| Mid | Skill | "Handle upsell offer", "Answer FAQ", "Generate review request" |
| Top | Agent | Support Agent, Sales Agent, Concierge Agent |

- **Tools** are atomic, discrete functions that are stateless, low-level, and reusable.
- **Skills** are packaged business-process capabilities that orchestrate multiple tools.
- **Agents** are role-based, autonomous AI workers with a defined persona, goal, skills, and judgment.

### Commercial Vocabulary (Operator-Facing)

- **Pre-built Agents**: Eleven core agents fully managed by Maison
- **Custom Agents**: Created by operators, no coding required
- **Mini-agents**: Single trigger + action (e.g., "text guests parking info 2 hours before arrival")

**Rule of thumb:**
- Internally (engineering, documentation): Tool → Skill → Agent
- Externally (sales, marketing): Pre-built Agent → Custom Agent → Mini-agent

### Maison Agent Stack Vision

Maison provides a modular platform of AI agents to manage hotel operations end to end, covering distribution, sales, marketing, revenue, intelligence, and guest communications. The platform starts with a free FAQ webchat and scales to a full commercial platform, giving independent hotels the capabilities of large chains without relying on legacy systems or OTAs.
