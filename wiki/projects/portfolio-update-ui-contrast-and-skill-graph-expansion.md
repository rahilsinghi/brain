# Portfolio Update: UI Contrast and Skill Graph Expansion

This update to the `rahilsinghi/portfolio` repository focuses on enhancing the user interface, specifically improving contrast in light mode. It also significantly expands the skill graph by integrating new project nodes and corresponding technology nodes, along with their interconnections.

## Key Concepts

- Light Mode UI Enhancements,- Skill Graph Updates,- Contrast Ratio,- Project Nodes,- Technology Nodes,- CSS Variables,- Knowledge Graph

## Details

This commit (`ca944fc`) on April 8, 2026, authored by Rahil Singhi, introduces substantial improvements to the `rahilsinghi/portfolio` project. It involved changing 6 files, with 127 additions and 42 deletions.

### Light Mode Enhancements
The user interface in light mode received several contrast and readability boosts:
-   **CSS Variable Adjustments:** `muted` and `border` CSS variables were darkened to ensure visible section dividers and text.
-   **Accent Green Deepening:** The primary accent green color was changed from `#00802A` to a deeper, more vibrant `#006B22`.
-   **Ticker Card Inversion:** Ticker cards were inverted to feature a black background with light text, improving contrast significantly on light pages.
-   **KnowledgeGraph Label Fixes:** Hardcoded color values (`#F0EDE6`) for KnowledgeGraph canvas labels were replaced with theme-aware implementations.
-   **Graph Visuals:** Edge alpha and node outlines were boosted within light mode for better visibility.
-   **Text Opacity:** The opacity of hackathon detail text was increased from `/50-/65` to `/70-/90`.

### Skill Graph Expansion
The skill graph, a key feature of the portfolio, was significantly expanded:
-   **New Project Nodes:** The following projects were added to the graph: Karen, Flock, Ask NYC, and MarkPush.
-   **New Technology Nodes:** A range of new technologies were incorporated, including ElevenLabs, Google ADK, Vercel AI SDK, Gemini Vision, Twilio, and Next.js.
-   **Interconnections:** All newly added project nodes were wired to their relevant technology nodes, and cross-skill relationships were established to enrich the graph's data.
