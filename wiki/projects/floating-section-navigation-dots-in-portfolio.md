# Floating Section Navigation Dots in Portfolio

This feature introduces floating navigation dots on the right side of the portfolio, which dynamically highlight the current section as the user scrolls. Users can click these dots for quick navigation, and section labels appear on hover. The feature is designed for larger screens and is hidden on mobile devices.
