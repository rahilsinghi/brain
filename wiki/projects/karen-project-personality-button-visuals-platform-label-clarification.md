# Karen Project: Personality Button Visuals & Platform Label Clarification

This update to the Karen project enhances the user interface by making personality selection buttons more visually interactive with glow effects, highlights, emojis, and a 'SELECTED' badge. Additionally, it clarifies the platform dropdown label to "Where Were You Ghosted?" for better user understanding.

## Details

This commit (`f8f70ce`) in the `rahilsinghi/karen` repository, authored by Rahil Singhi on 2026-04-04, focuses on improving the frontend user experience.

Key changes include:

*   **Personality Buttons**: Enhanced visual feedback for personality selection buttons. Selected buttons now feature a glow border, a pink highlight, an emoji, and a 'SELECTED' badge to clearly indicate their active state.
*   **Hover State**: Unselected personality buttons now display a border hint upon hover, guiding users to potential selections.
*   **Platform Dropdown Label**: The label for the platform dropdown menu has been updated from a potentially ambiguous phrasing to the more descriptive and user-friendly "Where Were You Ghosted?".

These changes aim to make the user interface more intuitive and engaging, particularly around the core interaction points of the Karen project.
