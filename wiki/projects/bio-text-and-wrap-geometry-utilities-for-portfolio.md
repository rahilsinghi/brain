# Bio Text and Wrap Geometry Utilities for Portfolio

This commit introduces new utilities for handling editorial content and geometric text wrapping within the `rahilsinghi/portfolio` project. It includes `bio.ts` for managing bio-related editorial content, intended for a Pretext background layer, and `wrap-geometry.ts` for calculating text routing around obstacles using circle/rectangle interval mathematics.

## Details
