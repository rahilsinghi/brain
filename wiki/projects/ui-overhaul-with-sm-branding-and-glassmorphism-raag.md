# UI Overhaul with SM Branding and Glassmorphism (raag)

This commit introduces a comprehensive frontend redesign for the `raag` project, adopting a red, white, and black color palette derived from Seedhe Maut branding. The update features glassmorphism cards with premium hover effects, custom typography using KinetikaMaut and TTCommonsPro, and integrated Seedhe Maut visual elements.

## Key Concepts

Frontend Redesign,SM Branding,Glassmorphism,User Interface (UI),Typography,Hover Effects,Color Palette

## Details

The `raag` project underwent a significant UI overhaul, aligning its aesthetic with Seedhe Maut (SM) branding. The new color scheme prominently features `#d91d1c` (red), `#050505` (black), and white. Headings now use the KinetikaMaut display font, while body text employs TTCommonsPro.

A key design element is the implementation of glassmorphism cards, which include premium hover effects to enhance user interaction. Visual components directly reflecting the Seedhe Maut brand include an album art carousel for empty states, the SM logo serving as the assistant's avatar, and album thumbnails within song cards. Furthermore, score rings and energy bars are accented in red, consistent with the brand's primary color.
