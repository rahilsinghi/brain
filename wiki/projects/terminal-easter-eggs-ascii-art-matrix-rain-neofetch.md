# Terminal Easter Eggs: ASCII Art, Matrix Rain, Neofetch

This commit introduces several terminal-based easter eggs and enhancements to the `rahilsinghi/portfolio` project. Users can now experience an ASCII art welcome message, a matrix rain effect, and system information via `neofetch` directly within the terminal.

ASCII art,Matrix rain effect,Neofetch,Terminal Easter eggs,Command-line interface (CLI)

## Details

This feature, implemented in the `rahilsinghi/portfolio` repository (SHA: `9104bf3`) on 2026-03-11, enhances the user experience with several interactive terminal elements.

Key additions include:

*   **ASCII Art Welcome Message**: A custom ASCII art message is now displayed upon opening the terminal, adding a personalized touch.
*   **Matrix Rain Effect**: A new command, `matrix`, triggers a visual effect reminiscent of the 'matrix rain' from the movie, simulating falling green characters on the terminal.
*   **Neofetch Integration**: The `neofetch` command is now available, allowing users to display system information and an ASCII logo in a stylish, condensed format.
*   **Denied `sudo rm -rf`**: An easter egg response is triggered if a user attempts the `sudo rm -rf /` command, playfully denying the destructive action.

These additions contribute to the interactive and engaging nature of the portfolio's terminal interface, showcasing technical creativity and attention to detail.
