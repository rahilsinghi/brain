# Add Original LaTeX Resume Source

This commit integrates the original LaTeX master template of the resume into the Career-Datacenter project. It serves as a foundational source for data extraction while preserving the professional formatting and structure for future use in generating a structured YAML database.

## Details

The `rahil_singhi_latex_master.tex` file, originally from Overleaf, has been added to the repository. This comprehensive LaTeX template includes all standard resume sections and provides flexibility through commented-out content. It leverages `fontawesome5` icons and custom resume macros to maintain a polished, professional appearance.

The primary purpose of integrating this LaTeX source is to provide a complete, authoritative base from which resume data can be extracted. This process aids in building a structured YAML database, ensuring that the integrity and original formatting of the resume are preserved.
