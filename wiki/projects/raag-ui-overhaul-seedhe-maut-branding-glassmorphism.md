# Raag UI Overhaul: Seedhe Maut Branding & Glassmorphism

This article details a comprehensive frontend redesign for the 'raag' project, implementing a striking red/white/black color scheme inspired by Seedhe Maut branding. Key features include glassmorphism cards, custom typography, and enhanced visual elements like album art carousels and detailed song cards.

## Key Concepts

- UI/UX Design,- Branding,- Glassmorphism,- Frontend Development,- Typography

## Details

A complete frontend redesign of the 'raag' application was implemented, centered around the distinctive Seedhe Maut brand palette. This palette utilizes a specific red (`#d91d1c`), black (`#050505`), and white for a bold and consistent aesthetic.

Typography was carefully chosen to complement the branding, with the 'KinetikaMaut' display font designated for headings and 'TTCommonsPro' for body text. A prominent visual feature of this overhaul is the adoption of glassmorphism for interactive cards, complete with premium hover effects to enhance user engagement.

Several specific UI elements were updated or introduced:
- An album art carousel fills the empty state, providing visual interest.
- The Seedhe Maut (SM) logo serves as the assistant avatar.
- Album thumbnails are now integrated into song cards for richer visual context.
- Score rings and energy bars feature a distinct red accent, aligning with the overall color scheme.
