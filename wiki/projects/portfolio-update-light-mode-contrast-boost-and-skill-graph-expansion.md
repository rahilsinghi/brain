# Portfolio Update: Light Mode Contrast Boost and Skill Graph Expansion

This commit to the [[Rahil Singhi's Portfolio]] repository significantly improves the light mode user interface by boosting contrast and readability across various elements. It also expands the [[Skill Graph]] by integrating nodes for recent projects like [[Karen Project]], [[Flock Project]], [[Ask NYC]], and [[MarkPush]], alongside new technology dependencies.

## Key Concepts

UI/UX Improvement,Light Mode,Contrast,Skill Graph,Project Integration,Technology Stack Updates,Portfolio Management

## Details

This update, commit `ca944fc` to the `rahilsinghi/portfolio` repository, focuses on two main areas: enhancing the light mode user interface and expanding the integrated [[Skill Graph]].

### Light Mode UI Enhancements

Several adjustments were made to improve contrast and visibility for users in light mode:

*   **CSS Variables Darkening**: `muted` and `border` CSS variables were darkened to ensure section dividers and text are more visible.
*   **Accent Green Deepening**: The primary accent green color was changed from `#00802A` to `#006B22` for increased vibrancy and better contrast.
*   **Ticker Card Inversion**: [[Featured Project Cards]] (ticker cards) now feature a black background with light text to provide strong contrast against the light-themed page.
*   **Knowledge Graph Label Fix**: Hardcoded labels in the [[Knowledge Graph]] canvas, previously `#F0EDE6`, are now dynamically themed to ensure they are visible in light mode.
*   **Graph Element Visibility**: The alpha of graph edges and the visibility of node outlines were boosted in light mode for clearer representation.
*   **Hackathon Detail Text Opacity**: The opacity of hackathon detail text was increased from `/50-/65` to `/70-/90` for improved readability.

### Skill Graph Expansion

The [[Skill Graph]] has been updated to reflect recent project and technology integrations:

*   **New Project Nodes**: The [[Karen Project]], [[Flock Project]], [[Ask NYC]], and [[MarkPush]] projects have been added as new nodes to the graph.
*   **New Technology Nodes**: Corresponding technology dependencies, including [[ElevenLabs]], [[Google ADK]], [[Vercel AI SDK]], [[Gemini Vision]], [[Twilio]], and [[Next.js]], have also been added as nodes.
*   **Interconnections**: All new project nodes are now wired to their respective technology nodes, establishing `project→tech` edges, and cross-skill relationships have been updated to reflect these additions.
