## Details

This significant update to the `rahilsinghi/dotfiles` repository (SHA: `aab3c96`) integrates advanced automation and AI-powered assistance into the development environment. The changes encompass new skills, custom agents, and a series of hooks to enhance productivity and code quality.

### New Skills
Three new command-line interface (CLI) skills have been added:
*   `scaffold-next`: For generating project scaffolds.
*   `env-check`: To validate environment configurations.
*   `deploy`: For simplified deployment processes.

### Custom Agents
Four custom AI agents are introduced to provide specialized assistance:
*   `backend-builder`: Focuses on backend development tasks.
*   `frontend-rapid`: Accelerates frontend development workflows.
*   `security-reviewer`: Aids in reviewing code for security vulnerabilities.
*   `test-writer`: Assists in generating comprehensive test cases.

### System Hooks
Six new hook scripts are implemented to automate and enforce development best practices:
*   `secret blocker`: Prevents accidental commitment of sensitive information.
*   `bash validator`: Ensures the validity of Bash scripts.
*   `prompt amplifier (/+)`: Enhances AI prompt interactions.
*   `ESLint on write`: Automatically runs ESLint upon file save.
*   `session init`: Handles session initialization tasks.
*   `completion verifier`: Verifies task completion.

### Documentation and Configuration Updates
*   `SETUP.md`: A comprehensive reference document detailing the entire setup.
*   `CLAUDE.md`: Updated to reflect all the new capabilities and their usage.
*   `settings.json`: Configured to wire up all the new hook scripts.
*   `.gitignore`: Modified to exclude ephemeral directories (`file-history`, `image-cache`, `ide`, `telemetry`) from version control.

This initiative aims to create a more robust, automated, and intelligent development environment, leveraging AI for various stages of the software development lifecycle. The `prompt amplifier` hook, in particular, signifies a deeper integration with AI models for enhanced human-AI collaboration.
