# Hero Parallax and Aurora Gradient Background Feature

This feature introduces a dynamic hero section with parallax scrolling text layers and a subtle, animated aurora-style gradient background. It enhances the visual depth and interactivity of the portfolio's main display.

## Key Concepts

- Parallax Scrolling,- CSS Gradients,- Web Animation,- Front-end Development,- User Interface (UI) Design

## Details

This commit (`606b934`) implements a new visual feature for the hero section of the `rahilsinghi/portfolio` project. It introduces several 'parallax layers' where text lines move at varying speeds during scrolling, creating an illusion of depth.

In addition, an animated gradient mesh, styled to resemble an aurora, has been added behind the hero text. This background effect is achieved using CSS-only techniques and maintains a low opacity to avoid overwhelming the foreground content. The overall goal is to provide a more engaging and visually rich initial experience for visitors to the portfolio.
