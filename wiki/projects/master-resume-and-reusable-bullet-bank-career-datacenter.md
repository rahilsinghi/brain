# Master Resume and Reusable Bullet Bank (Career-Datacenter)

This update to the `Career-Datacenter` repository introduces a comprehensive master resume and a categorized bullet bank. These resources are designed to streamline the process of rapidly tailoring resumes for various target roles, including AI/ML, backend, data, and product positions.

- [[Master Resume]],- [[Bullet Bank]],- [[Resume Tailoring]],- [[Career-Datacenter]]

## Details

This commit (`17cf3a7`) by Rahil Singhi on 2026-02-10 adds two crucial documentation files to the `rahilsinghi/Career-Datacenter` repository:

### `master_resume_source.md`
This file contains a complete resume encompassing all bullets from past experiences and projects. It includes specific tailoring notes for different role types, such as AI/ML, backend, data, and product management. Each position is detailed with relevant tech stack summaries. The master resume covers 5 projects and 4 experiences in full detail.

### `bullet_bank.md`
This document provides over 41 reusable bullet points, categorized for easy access and application. Categories include:
*   AI/ML
*   Backend
*   Data
*   Cloud
*   SEO
*   Documentation
*   Leadership
*   Research

Each bullet is linked to an evidence ID, ensuring traceability and context. The bullet bank also features a quality checklist and usage instructions, ensuring effective and consistent application of these tailored statements. Together, these documents enable efficient and high-quality resume generation tailored to specific job requirements.
