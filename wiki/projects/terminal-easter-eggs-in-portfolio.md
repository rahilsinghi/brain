# Terminal Easter Eggs in Portfolio

This feature introduces several interactive 'easter egg' commands and an ASCII art welcome message to the terminal interface of the user's portfolio website. It includes commands for a matrix rain animation, system information display, and a humorous 'sudo rm -rf' response.

## Key Concepts

[[Terminal Easter Eggs]],[[ASCII Art]],[[Matrix Rain]],[[Neofetch]],[[Command Line Interface (CLI)]]

## Details

Implemented as part of the `rahilsinghi/portfolio` project (SHA: `9104bf3`), this enhancement adds several engaging features to the embedded terminal:

*   **ASCII Art Welcome:** An ASCII art message is now displayed automatically upon opening the terminal, providing a custom greeting.
*   **`matrix` command:** Typing `matrix` into the terminal initiates a classic matrix rain animation.
*   **`neofetch` command:** This command, inspired by the popular system information tool, displays simulated system details.
*   **`sudo rm -rf` command:** A humorous response is provided when a user attempts to execute this destructive command, indicating that the action is denied.

These additions aim to enhance user engagement and provide a personalized, interactive experience within the portfolio's terminal environment.
