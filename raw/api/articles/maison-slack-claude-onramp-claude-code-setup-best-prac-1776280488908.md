## Prompting Best Practices (Sandeep, April 13)
- Don't be verbose. Be specific.
- Don't expect Claude to figure out intent. Be clear and concise.
- Add screenshots when Claude fails with code assumptions
- Ask Claude to open browser and run tests for UI assumptions
- Compare browser against screenshot for better work
- If Claude hasn't delivered in 2 sessions, clear chat and start fresh
- If still failing, go to Claude web/Gemini/ChatGPT to diagnose why
- Run one task at a time. Fresh session per topic change.

## Dev Service Management (Sandeep, April 15)
- Start services: "start all dev services and control the logs"
- Claude maintains PIDs and logs in .local dir
- When issues arise, Claude already has access to logs
